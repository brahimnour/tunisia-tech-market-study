# Étude de marché — Secteur tech tunisien

## Description
Analyse du marché tech tunisien pour identifier les secteurs porteurs 
et les profils les plus demandés en BI & Data Analytics.
Données collectées sur 15 entreprises tech tunisiennes réelles.

## Compétences utilisées
- Collecte et structuration de données
- Python (pandas, matplotlib, seaborn)
- Analyse exploratoire de données (EDA)
- Rédaction d'un rapport structuré avec recommandations

## Résultats clés
- SQL et Python sont les technologies les plus demandées
- Le Consulting IT concentre le plus d'opportunités BI
- Plus de 70% des entreprises proposent des stages
- Data Analyst est le profil le plus recherché
