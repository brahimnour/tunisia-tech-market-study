# Rapport — Étude de marché du secteur tech tunisien

**Auteure :** Nour Brahim  
**Date :** 2026  
**Objectif :** Identifier les secteurs porteurs et les profils les plus 
demandés en BI & Data Analytics en Tunisie

---

## 1. Contexte

Dans le cadre d'une démarche d'orientation professionnelle en Business 
Intelligence, cette étude analyse le marché tech tunisien afin de mieux 
comprendre les besoins des entreprises en termes de compétences data.

---

## 2. Méthodologie

- Collecte manuelle de données sur 15 entreprises tech tunisiennes
- Structuration des données dans un fichier CSV
- Analyse exploratoire avec Python (pandas, matplotlib, seaborn)
- Synthèse des résultats sous forme de recommandations

---

## 3. Résultats clés

### 3.1 Secteurs dominants
Le **Consulting IT** concentre le plus grand nombre d'entreprises tech, 
suivi par la **Fintech** et le **Software**. Ces secteurs recrutent 
activement des profils BI & Data.

### 3.2 Technologies les plus demandées
| Technologie | Popularité |
|---|---|
| SQL | Très demandé |
| Python | Très demandé |
| Power BI | Demandé |
| Tableau | Présent |

### 3.3 Profils les plus recherchés
- **Data Analyst** — profil le plus demandé toutes entreprises confondues
- **BI Developer** — fort besoin dans les grandes entreprises
- **BI Consultant** — principalement dans le Consulting IT

### 3.4 Disponibilité des stages
Plus de **70% des entreprises** analysées proposent des stages, ce qui 
confirme l'accessibilité du marché pour les étudiants en BI.

---

## 4. Recommandations

1. **Prioriser SQL et Python** — technologies incontournables sur le marché tunisien
2. **Cibler le Consulting IT et la Fintech** — secteurs les plus actifs en recrutement BI
3. **Se concentrer sur Tunis** — ville qui concentre la majorité des opportunités
4. **Viser le profil Data Analyst** — le plus accessible en début de carrière

---

## 5. Conclusion

Cette étude confirme que le marché tech tunisien offre de réelles 
opportunités pour les profils BI & Data Analytics. Les compétences en 
SQL et Python sont indispensables, et le secteur Consulting IT représente 
le principal vivier de stages et d'emplois dans ce domaine.
